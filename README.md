# 1D UV Square

Blender add-on.

Create initial UV-quads around 3 selected vertices of a mesh.

Add-on functionality
-
Creates new quads around 3 selected vertices on a mesh. Quad is placed in the same plane as the points. Quad size is equivalent of the UV-initial space ((0.0, 0.0), (1.0, 1.0)).
